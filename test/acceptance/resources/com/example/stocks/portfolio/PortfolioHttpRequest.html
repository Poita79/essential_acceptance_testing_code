Portfolio Valuation

HTTP Adapter to Java Message (Request) Test

Implementation notes

The Portfolio's adapter is implemented in terms of HTTP; it's a RESTful server which collaborates with this business interface directly. It turns the HTTP API into a Java API; we're testing that a HTTP message turns into a Java message.

Given

A customer book containing 100 shares in AMZN and yesterday's stock price in the market data system for AMZN is 261.82

When

The user requests a valuation (the portfolio receives a request for a valuation of the client's book)

Then

The total portfolio valuation will be 26,185.00
